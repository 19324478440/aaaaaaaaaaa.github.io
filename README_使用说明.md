# MathMaster 使用说明

## 快速开始

### 方法一：使用启动脚本（推荐）

1. **双击运行 `启动服务器.bat`**
2. 等待服务器启动（会显示 "MathMaster 后端服务器启动成功！"）
3. **打开浏览器，访问：`http://localhost:3000`**
4. 使用以下账号登录：
   - 用户名：`demo`
   - 密码：`demo`（或留空）
   - 或直接点击"快速体验"按钮

### 方法二：手动启动

1. **打开命令提示符（CMD）或 PowerShell**
2. **进入项目目录**：
   ```bash
   cd d:\999\7
   ```

3. **安装依赖**（如果还没有安装）：
   ```bash
   npm install
   ```

4. **启动服务器**：
   ```bash
   node server.js
   ```
   或使用：
   ```bash
   npm start
   ```

5. **打开浏览器，访问：`http://localhost:3000`**

## 重要提示

⚠️ **不能直接双击 HTML 文件运行！**

原因：
- 前端代码需要连接到后端 API 服务器
- 直接打开 HTML 文件（file:// 协议）会遇到 CORS 跨域问题
- 服务器需要运行在 `http://localhost:3000` 才能提供 API 服务

## 停止服务器

在运行服务器的命令行窗口中，按 `Ctrl + C` 即可停止服务器。

## 常见问题

### 1. 提示 "node 不是内部或外部命令"
**解决方案**：需要先安装 Node.js
- 访问 https://nodejs.org/ 下载并安装
- 安装时务必勾选 "Add to PATH" 选项
- 安装后重新打开命令行窗口

### 2. 提示端口被占用
**解决方案**：
- 检查是否有其他程序占用 3000 端口
- 或者修改 `server.js` 中的 `PORT` 变量为其他端口（如 3001）

### 3. 登录后显示错误
**解决方案**：
- 确保服务器正在运行
- 检查浏览器控制台是否有错误信息
- 确认访问的是 `http://localhost:3000` 而不是直接打开 HTML 文件

## 技术栈

- **前端**：HTML + CSS + JavaScript
- **后端**：Node.js + Express
- **认证**：JWT (JSON Web Token)
- **数据存储**：内存存储（开发版本）

## 开发说明

如需修改代码：
- **前端代码**：编辑 `index.html`
- **后端代码**：编辑 `server.js`
- 修改后需要重启服务器才能生效
